# Zweitstimme Polling API

Welcome to the Zweitstimme Polling API documentation.

## Overview

The Zweitstimme Polling API provides access to German election polling data, scraped from various sources and normalized for analysis.

## Features

- **Real-time Data**: Latest polling data from major institutes
- **Historical Data**: Access to historical polling trends
- **RESTful API**: Clean, well-documented endpoints
- **Multiple Formats**: JSON, CSV, and Parquet exports

## Quick Start

```bash
# Install dependencies
uv sync

# Run the API
pollingapi server:start
```
